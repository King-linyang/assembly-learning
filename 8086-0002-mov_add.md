# mov

| 汇编指令      | 控制CPU完成的操作 | 高级语言语法描述 |
|-----------|------------|----------|
| mov ax,18 | 将18送入ax    | AX=18    |
| mov ah,18 | 将18送入ah    | AH=18    |
| add  ax,8 | 将8加到ax中    | AX=AX+8  |
| mov ax,bx | 将bx的值送入ax  | AX=BX    |
| add ax,bx | 将bx加到ax中   | AX=AX+BX |

`汇编指令不区分大小写`  
mov ax,18  
这么写默认是十进制  
如果是16进制,需要在最后加H  
mov ax,18H
